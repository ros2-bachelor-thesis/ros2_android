# Object Detection rviz2 Testing

Visualize 3D detection output from the Android perception pipeline using rviz2.

## Prerequisites

- ROS 2 Humble
- rviz2
- ZED rosbag with RGB, depth, and point cloud topics
- Android device running the ros2_android app on the same DDS domain

## Topics

Published by the Android app:

| Topic                | Type                      | Description                |
| -------------------- | ------------------------- | -------------------------- |
| `/cpb_beetle_center` | `geometry_msgs/Point`     | Beetle 3D center           |
| `/cpb_larva_center`  | `geometry_msgs/Point`     | Larva 3D center            |
| `/cpb_eggs_center`   | `geometry_msgs/Point`     | Eggs 3D center             |
| `/cpb_beetle`        | `sensor_msgs/PointCloud2` | Beetle cropped point cloud |
| `/cpb_larva`         | `sensor_msgs/PointCloud2` | Larva cropped point cloud  |
| `/cpb_eggs`          | `sensor_msgs/PointCloud2` | Eggs cropped point cloud   |

From rosbag:

| Topic                                        | Type                      | Description    |
| -------------------------------------------- | ------------------------- | -------------- |
| `/zed/zed_node/point_cloud/cloud_registered` | `sensor_msgs/PointCloud2` | Full ZED scene |

## Running

```bash
# Terminal 1: Play ZED rosbag
ros2 bag play /path/to/rosbag --loop --rate 0.1

# Terminal 2: Android app already running on device (connected via Wi-Fi DDS)

# Terminal 3: Start marker bridge (converts Point -> Marker for rviz2)
python3 ros2_android/scripts/tests/object_detection/perception_rviz_markers.py

# Terminal 4: Launch rviz2
rviz2
```

## rviz2 Setup

1. Set **Fixed Frame** to `zed_left_camera_frame`
2. Add displays:
   - **PointCloud2** - topic: `/zed/zed_node/point_cloud/cloud_registered` (base scene, white/intensity color)
   - **MarkerArray** - topic: `/cpb_detections_markers` (detection center points as colored spheres)
   - **PointCloud2** - topic: `/cpb_eggs` (optional, cropped cloud per detection)
   - **PointCloud2** - topic: `/cpb_beetle` (optional)
   - **PointCloud2** - topic: `/cpb_larva` (optional)

## Marker Colors

- Green spheres: `cpb_beetle`
- Red spheres: `cpb_larva`
- Blue spheres: `cpb_eggs`

Markers auto-expire after 1 second.

## Validation

To compare C++ 3D output against the Python reference, capture the VALIDATION_3D log:

```bash
adb logcat | grep VALIDATION_3D
```

Format: `VALIDATION_3D,x1,y1,x2,y2,confidence,class_id,class_name,point3d_x,point3d_y,point3d_z`

Compare against `~/Documents/bachelor-thesis/validation_data/detections_3d.csv` generated by the Python `object_detection` node.
